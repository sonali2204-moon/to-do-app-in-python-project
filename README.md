# to-do-app-in-python-project
Python Task Management System (To-Do App)
📌 Project Overview
This is a command-line interface (CLI) application built using Python that allows users to manage their daily tasks efficiently. The program demonstrates fundamental programming concepts such as list manipulation, loops, conditional logic, and user input handling.

This project was developed as part of my Digital Literacy (CSE0001) portfolio to showcase technical proficiency in Python and version control using GitHub.

🚀 Features
Bulk Initialization: Add multiple tasks at once when starting the program.

Add Tasks: Dynamically add new items to your list.

Update Tasks: Modify existing tasks if plans change.

Delete Tasks: Remove completed or unnecessary tasks from the list.

View List: Display all current tasks at any time.

Persistence: The program runs in a continuous loop until the user chooses to exit.

🛠️ Technologies Used
Language: Python 3.x

Key Concepts: * Lists (CRUD operations)

While and For loops

Conditional Statements (if-elif-else)

Functional Programming

📋 How to Run
Ensure you have Python installed on your system.

Copy the code into a file named app.py.

Open your terminal or command prompt.

Run the script using:

Bash
python app.py
📸 Usage Example
Start: Enter the number of tasks (e.g., 2).

Input: Enter "Study Physics" and "Finish Lab Report".

Manage: Enter 1 to add a new task, or 3 to delete one.

Exit: Enter 5 to safely close the application.
